# Disavo-Tools-2.0
Wewnętrzne narzędzie SEO Sempai do weryfikacji i generowania plików disavow. Sprawdza aktywność domen i podstron przez DNS over HTTPS, importuje dane z Ahrefs CSV, a następnie generuje gotowy plik disavow.txt zgodny z wytycznymi Google Search Console. Działa w całości po stronie przeglądarki — bez backendu.
